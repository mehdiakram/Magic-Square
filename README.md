# Magic Square 🧩

**Magic Square** is an interactive web application based on the legendary Indian mathematician **Srinivasa Ramanujan's** birthday magic square. It takes a user's date of birth and generates a unique 4x4 grid where the sum of numbers in every row, column, and diagonal is always equal.

---

## 🌟 Features

- **Personalized Magic Square:** Generate a unique magic square based on your specific birth date.
- **Real-time Proof:** Displays the live sum for every row and column to prove the mathematical consistency.
- **Vertical Visuals:** Column sums are displayed in a modern vertical style for better visibility and aesthetics.
- **Interactive Highlighting:** Includes buttons to instantly highlight different patterns (Corners, Center 2x2, Quadrants) that form the magic sum.
- **Modern UI:** A clean, responsive, and user-friendly interface built with a professional look.

---

## 📖 How to Use

1. Clone the repository or download the `index.html` file.
2. Open the file in any modern web browser.
3. Select your date of birth from the date picker.
4. Click the **"Generate Square"** button to see the magic!
5. Use the highlight buttons to explore different combinations that add up to your Magic Total.

---

## 📐 The Formula

This project utilizes the famous formula discovered by Ramanujan:
- **Row 1:** $A, B, C, D$ (Representing DD-MM-CC-YY)
- **Row 2:** $D+1, C-1, B-3, A+3$
- **Row 3:** $B-2, A+2, D+2, C-2$
- **Row 4:** $C+1, D-1, A+1, B-1$
